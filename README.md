## Setup Instructions
### Backend
1. cd backend
2. pip install -r requirements.txt
3. Create .env file (see .env.example)
4. python -m uvicorn app.main:app --reload

### Frontend
1. cd frontend
2. npm install
3. Create .env.local (see .env.example)
4. npm run dev
